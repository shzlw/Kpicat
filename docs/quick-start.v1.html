Create an account

1. Go to the sign up page and enter the account information.

  • Corporation will be used later when log into mobile app.

Create a page

1. Choose the Page tab and click Add button.

2. Enter the Page Name and configure the layout.

  • Page Name is also used in the mobile app as menu item.

  • Title Color defines the tool bar color in the moible app when this page is selected.

3. Select the number of columns and then click Add Row button. A row will be added to the layout.

  • The page layout is configured based on rows.

  • Each row can have up to 5 columns.

  • Each column is a component.

4. Drag and Drop the row if it needs to be repositioned. Position of columns cannot be changed.

  • A row can be removed by selecting any columns within the row and then click Remove Row button. The Entire Row will then be removed.

5. Click Save button to save the page. Then click the Component again to view the Component Id.

  • Component Id will be used later in the REST api.

Create a role

1. Choose the Role tab and click Add button.

2. Enter Role Name and add Page to it.

Create a user

1. Choose the User tab and click Add button.

2. Enter the Username, Password and assign a Role.

3. The Registered Account can also be assigned a Role in the Config tab.

Customize the app

1. Choose the Config tab and click Mobile Customization.

2. Change the Colors and click Save button.

Call REST api

1. Check the API document to send data.

Download the mobile app

1. Go to the Google Play Store to download the Android App.

View the data

1. Login and select the page you just created!

FAQ

Why cannot I see pages on my mobile app?

Normally this is caused by misconfiguration of roles. Please check if pages are added to the role and if the user is assigned a role.
